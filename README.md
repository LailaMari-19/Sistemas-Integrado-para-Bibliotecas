# Sistema Integrado para Bibliotecas

O projeto tem o principal objetivo de elaborar uma solução que permita maior agilidade nos processos de cadastro de livros, cadastro de usuários, e empréstimo de exemplares em uma biblioteca. 
A solução desenvolvida visa proporcionar aos leitores uma experiência mais agradável e eficiente ao acessar seus gêneros literários favoritos, facilitando a busca, e o empréstimo de livros. Além disso, busca incentivar visitas futuras à biblioteca, promovendo maior engajamento dos usuários e o aproveitamento contínuo do acervo disponível.
